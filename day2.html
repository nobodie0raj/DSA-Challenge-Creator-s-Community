Day 2: Exploring Arrays and Linked Lists

Arrays

Arrays are like rows of lockers where each locker can store one item, and you can find any item by knowing its locker number (index).

Example:

Fruits: ["Apple", "Banana", "Cherry", "Date", "Elderberry"]
Index:  [0,       1,       2,       3,      4]

Key Points:

  • Arrays store items in a fixed order.
  • Accessing an item is fast because you can directly use its index.
  • Inserting or deleting an item can be slow because you might need to move other items. Types of Arrays

Simple Python Example:


fruits = ["Apple", "Banana", "Cherry"]
print(fruits[1])  # Output: Banana

Output:

Banana

Linked Lists

Linked lists are like a treasure hunt where each clue leads you to the next one. Each item (node) points to the next item in the list.

Example:

Head -> [1] -> [2] -> [3] -> [4] -> None

Key Points:

  • Linked lists can grow and shrink easily because each item points to the next one.
  • Accessing an item can be slow because you need to follow the pointers from the start.
  • Inserting or deleting an item is fast because you only need to change the pointers. Types of Linked List

Simple Python Example:


class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

head = Node(1)
second = Node(2)
third = Node(3)

head.next = second
second.next = third

# Printing the linked list
current = head
while current:
    print(current.data)
    current = current.next
# Output: 1 2 3

Output:

1
2
3

Simple C Program to Create and Print a Linked List with 3 Nodes

Step 1: Define the Node Structure


        // Define the structure for a node in the linked list
        struct Node {
            int data; // The data stored in the node
            struct Node* next; // Pointer to the next node in the list
        };

Step 2: Create the Nodes

        int main() {
            // Create 3 nodes
            struct Node* head = (struct Node*)malloc(sizeof(struct Node));
            struct Node* second = (struct Node*)malloc(sizeof(struct Node));
            struct Node* third = (struct Node*)malloc(sizeof(struct Node));

Step 3: Assign Data to Each Node


            // Assign data to each node
            head->data = 1;    // First node data
            second->data = 2;  // Second node data
            third->data = 3;   // Third node data

Step 4: Link the Nodes Together


            // Link the nodes together
            head->next = second;  // First node points to second node
            second->next = third; // Second node points to third node
            third->next = NULL;   // Third node is the last node, so it points to NULL

Step 5: Print the Linked List


            // Print the linked list
            struct Node* current = head; // Start from the head node
            while (current != NULL) {
                printf("%d ", current->data); // Print the data of the current node
                current = current->next; // Move to the next node
            }
            // Output: 1 2 3

Step 6: Free the Allocated Memory


            // Free the allocated memory for each node
            free(head);
            free(second);
            free(third);
        
            return 0;
        }
