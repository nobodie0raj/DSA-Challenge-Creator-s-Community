Day 1: Introduction to Data Structures and Algorithms (DSA)

What is a Data Structure?

A data structure is a way to organize and store data so that we can use it efficiently. Think of it like organizing your toys or books in a way that makes them easy to find and use.

Understanding Time Complexity

Imagine you have a big toy box full of different toys. Sometimes, you want to find your favorite toy, and other times, you want to clean up and put all the toys in order. How long it takes to do these things depends on the number of toys you have and how you do it. This is similar to what we call time complexity in computer science.

Example 1: Finding a Toy

Linear Search

Imagine you are looking for a specific toy in your toy box. You start from the first toy and check each one until you find the toy you're looking for. This is called linear search.

  • If you have 10 toys, you might have to look at up to 10 toys.
  • If you have 100 toys, you might have to look at up to 100 toys.

The time it takes increases as the number of toys increases. We call this O(n), where "n" is the number of toys.

Binary Search

Now, imagine your toys are already sorted in order, like a bookshelf. You can start in the middle and decide if you need to look to the left or right.

  • If you have 10 toys, you might only need to check 4 toys.
  • If you have 100 toys, you might only need to check 7 toys.

This method is much faster for large numbers of toys. We call this O(log n).

Common Data Structures

  • Array: Imagine a row of lockers. Each locker can hold one item, and you can find any item by knowing its locker number (index). Example: An array of favorite fruits: ["Apple", "Banana", "Cherry"]
  • Linked List: Think of a treasure hunt where each clue leads you to the next one. Each item (node) points to the next item in the list. Example: [1] -> [2] -> [3] -> [4]
  • Stack: Like a stack of plates where you can only add or remove the top plate. It follows the Last In, First Out (LIFO) principle. Example: A stack of books: [Book1, Book2, Book3] (you can only take the top book)
  • Queue: Like a line of people waiting to buy ice cream. The first person in line is served first. It follows the First In, First Out (FIFO) principle. Example: A queue of people: [Person1, Person2, Person3]
  • Tree: Imagine a family tree where each person (node) can have children (sub-nodes). It's a hierarchical structure. Example: 
            Grandpa
       /      \
     Mom      Uncle
    /  \        /  \
You Sibling Cousin1 Cousin2
  • Graph: Think of a map with cities (nodes) and roads (edges) connecting them. It's a network of connections. Example: 
    CityA --- CityB
  |        /
CityC --- CityD

What is an Algorithm?

An algorithm is a step-by-step process to solve a problem. It's like following a recipe to bake a cake.

Common Algorithms

  • Sorting: Organizing items in a certain order. Example: Arranging books from shortest to tallest. Bubble Sort: Compare each pair of adjacent items and swap them if they are in the wrong order. Repeat until the list is sorted.
  • Searching: Finding an item in a collection of items. Example: Looking for a book in a library. Linear Search: Check each item one by one until you find what you're looking for. Binary Search: If the list is sorted, you can quickly find an item by repeatedly dividing the list in half.
  • Traversal: Visiting all the nodes or items in a data structure. Example: Walking through all the rooms in a house. Tree Traversal: In a tree, you might visit nodes in a certain order (pre-order, in-order, post-order).

Let's Summarize

Data Structures: Ways to organize data (arrays, linked lists, stacks, queues, trees, graphs).

Algorithms: Step-by-step instructions to solve problems (sorting, searching, traversal).

Simple Example in Python


# An array of numbers
numbers = [5, 2, 9, 1, 5, 6]

# Sorting the array
sorted_numbers = sorted(numbers)

print("Original numbers:", numbers)
print("Sorted numbers:", sorted_numbers)

Output:

Original numbers: [5, 2, 9, 1, 5, 6]
Sorted numbers: [1, 2, 5, 5, 6, 9]

Simple Example in C


int main() {
    int numbers[] = {5, 2, 9, 1, 5, 6};
    int n = sizeof(numbers)/sizeof(numbers[0]);
    int i;
    
    bubbleSort(numbers, n);
    
    printf("Sorted numbers: ");
    for (i = 0; i < n; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    return 0;
}

Output:

Sorted numbers: 1 2 5 5 6 9
